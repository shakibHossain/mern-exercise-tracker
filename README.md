## Exercise Tracker

This project was built with MERN (MongoDB, Express, React and Node JS) stack.

Features -

1. Create user.
2. Create/Edit/Delete exercise logs.
3. Bootstrap used for front end.
4. MongoDB Atlas used for data storage.
